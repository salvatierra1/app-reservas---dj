# appEvents

1. Clone project
```https://github.com/javierCh77/appEvents```


2. Create environment variable
```
python -m venv venv
```


3. Run virtual environment
```
venv\Scripts\activate
```

4. Install Django in folder ```src/```
```
pip install Django
```
5. Create migration
```
python manage.py migrate
```
6. Runing server 
```
python manage.py runserver
```
